## study-session-picker
A simple web app that helps students:
- Add study tasks
- Randomly choose what to work on
- Use a built-in 25-minute focus timer
- Save tasks in the browser

## Features
- Task list with local storage
- Random task generator
- Pomodoro timer
- Clean aesthetic UI

## languages used
- HTML
- CSS
- JavaScript
